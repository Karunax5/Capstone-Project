Introduction

This site was created as the final project for Web Design for Everybody Capstone. It was built using the Bootstrap Framework. As a topic I chose my favorite dog breed, the Welsh Corgi. The next 2 pages detail the 2 main types of Corgis.

Each page contains 4 pictures each. Each picture is a link to the source of the image. In a viewing window of less than 576px, these images are displayed as an image carousel at the bottom of the page. Also, the navigation turns into a drop down menu in this viewing window. The banner picture will display the source of the image in the top right corner if it is hovered over. Finally, there is a sticky footer on each page.
